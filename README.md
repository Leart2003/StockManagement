

A web-based Stock Management System built with ASP.NET Core MVC that helps manage products, inventory, and basic business operations efficiently.

 Features

* **Product Management**

  * Add, update, delete, and view products
  * Store product details such as name, price, quantity, and description

* **Category Management**

  * Organize products into categories
  * Improve filtering and structure of inventory

* **Stock Control**

  * Track product quantities in real-time
  * Prevent stock inconsistencies

* **User Authentication**

  * Secure login and registration system using ASP.NET Identity
  * Only authorized users can manage stock

* **Wishlist Feature**

  * Users can save products to their personal wishlist

* **Order System**

  * Place orders directly from the product details page
  * Pre-filled order forms for better user experience

### 🛠️ Technologies Used

* ASP.NET Core MVC (.NET 8)
* Entity Framework Core
* SQL Server
* HTML, CSS, JavaScript
* Bootstrap
